779 /*************************************************************************
780 Integration on a finite interval [A,B].
781 Integrand have integrable singularities at A/B.
782 
783 F(X) must diverge as "(x-A)^alpha" at A, as "(B-x)^beta" at B, with known
784 alpha/beta (alpha>-1, beta>-1). If alpha/beta are not known, estimates
785 from below can be used (but these estimates should be greater than -1 too).
786 
787 One of alpha/beta variables (or even both alpha/beta) may be equal to 0,
788 which means than function F(x) is non-singular at A/B. Anyway (singular at
789 bounds or not), function F(x) is supposed to be continuous on (A,B).
790 
791 Fast-convergent algorithm based on a Gauss-Kronrod formula is used. Result
792 is calculated with accuracy close to the machine precision.
793

798  Beta - power-law coefficient of the F(x) at B,
