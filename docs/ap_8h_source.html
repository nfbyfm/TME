4556 /********************************************************************
4557 CSV operations: reading CSV file to real matrix.
4558 
4559 This function reads CSV file and stores its contents to double
4560 precision 2D array. Format of the data file must conform to RFC 4180
4561 specification, with additional notes:
4562 * file size should be less than 2GB
4563 * ASCI encoding, UTF-8 without BOM (in header names) are supported
4564 * any character (comma/tab/space) may be used as field separator, as
4565  long as it is distinct from one used for decimal point
4566 * multiple subsequent field separators (say, two spaces) are treated
4567  as MULTIPLE separators, not one big separator
4568 * both comma and full stop may be used as decimal point. Parser will
4569  automatically determine specific character being used. Both fixed
4570  and exponential number formats are allowed. Thousand separators
4571  are NOT allowed.
4572 * line may end with \n (Unix style) or \r\n (Windows style), parser
4573  will automatically adapt to chosen convention
4574 * escaped fields (ones in double quotes) are not supported
4575 
4576 INPUT PARAMETERS:
4577  filename relative/absolute path
4578  separator character used to separate fields. May be ' ',
4579  ',', '\t'. Other separators are possible too.
4580  flags several values combined with bitwise OR:
4581  * alglib::CSV_SKIP_HEADERS - if present, first row
4582  contains headers and will be skipped. Its
4583  contents is used to determine fields count, and
4584  that's all.
4585  If no flags are specified, default value 0x0 (or
4586  alglib::CSV_DEFAULT, which is same) should be used.
4587 
4588 OUTPUT PARAMETERS:
4589  out 2D matrix, CSV file parsed with atof()
4590 
4591 HANDLING OF SPECIAL CASES:
4592 * file does not exist - alglib::ap_error exception is thrown
4593 * empty file - empty array is returned (no exception)
4594 * skip_first_row=true, only one row in file - empty array is returned
4595 * field contents is not recognized by atof() - field value is replaced
4596  by 0.0
